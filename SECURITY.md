# Security Policy

## Reporting a Vulnerability

**Do not report vulnerabilities via public GitHub issues, discussions, or social media.**

If you discover a potential security or safety issue, please email:
**[ez945y@gmail.com](mailto:ez945y@gmail.com)**

### Please include:
- Description of the vulnerability and its potential safety impact.
- Steps to reproduce the issue (including relevant Stackfiles).
- Any suggested fixes or mitigations.

We will acknowledge your report within 48 hours and coordinate a fix and disclosure.
